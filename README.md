# onixlib - Bibliothèque de gestion de données pour ONIX

![PyPi version](https://img.shields.io/pypi/v/onixlib)
![Licence](https://img.shields.io/badge/license-MIT-green)
![Python version](https://img.shields.io/pypi/pyversions/onixlib)

**onixlib** est une bibliothèque Python souveraine dédiée à la lecture et au parsing des fichiers ONIX, et à la création de ceux-ci à partir de données structurées.
Elle fournit des modèles de données basés sur les spécifications ONIX, ainsi que des fonctions pour convertir entre les formats XML et les objets Python.

## 🚀 Objectifs

- Offir une **implémentation Python propre et moderne** des structures de données ONIX.
- Fournir des **dataclasses typées** pour chaque bloc (groupes) de données ONIX, facilitant ainsi la manipulation de données par blocs d'arborescence.
- Faciliter l'intégration du modèle de données ONIX dans des applications Python, en permettant une conversion facile entre Objets <--> XML.
- Proposer une base souveraine et open-source pour les acteurs du livre.

## ✨ Fonctionnalités

- 📦 Modèles de données ONIX 3.0
- 🧩 Parseur robuste
- 🔍 Validation des types et des valeurs sur le modèle ONIX
- 🧪 Tests unitaires inclus.
